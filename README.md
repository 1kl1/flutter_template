

# 시작하기

### 요구사항

- Flutter SDK 
- Dart SDK 
- FVM(Flutter Version Management)
- Android Studio / VS Code
- Git

### 설치 및 실행

1. 저장소 클론하기

```bash
git clone https://github.com/linkive/favy_app/
cd favy
```

2. FVM을 이용한 Flutter 버전 설정

3. 의존성 패키지 설치

```bash
flutter pub get
```
